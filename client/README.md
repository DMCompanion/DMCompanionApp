DevMtn Companion App
=====================

## Using this project

Run:

```bash
$ npm install
```

Then run:

```bash
$ ionic serve --port 9001
```

The node server should be run in a seperate terminal instance:

```bash
$ node index.js
```

Project Board: [Companion Trello](https://trello.com/b/kJjFH4UV/dm-companion)
